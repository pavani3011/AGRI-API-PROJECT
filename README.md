# Agri AI — Crop Recommendation System

## Overview
ML pipeline to recommend the best crop based on soil nutrients and climate.

## Dataset
- Source: Crop Recommendation Dataset (Kaggle)
- 2200 samples, 22 crop types, 7 features
- Features: N, P, K, temperature, humidity, ph, rainfall

## Week 1 Results (Baseline)
| Model               | Accuracy | F1 Score |
|---------------------|----------|----------|
| Logistic Regression | 96.78%   | 96.73%   |
| Decision Tree       | 98.85%   | 98.85%   |
